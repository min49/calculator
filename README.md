# Calculator

### Features
- Comma separated display
- 10 Digit limit (similar to physical calculator)

```
Buttons pressed     Calculation
 1 =                (=> 1)
 1 + =              (1 + 1 => 2)
 1 + = =            (1 + 1 => 2 + 1 => 3)
 1 + 2 = =          (1 + 2 => 3 + 2 => 5)
 1 + 2 = 4 =        (1 + 2 => 3, 4 + 2 => 6)
 1 + 2 + 4 =        (1 + 2 + => 3 + 4 => 7)
 1 + 2 = + 4 =      (1 + 2 => 3 + 4 => 7)
 1 + 2 = 4 + 5 =    (1 + 2 => 3, 4 + 5 = 9)
```
